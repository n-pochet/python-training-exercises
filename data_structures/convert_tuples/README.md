# Convert Tuples

Implement the `convert` function in `convert_tuples.py`
It should take a list of tuples as an input and convert it to a dictionary.

```
>>> l = [('a', 1), ('b', 2)]
>>> convert(l)
{'a': 1, 'b': 2}
```
